# Ceya

Simple toy programming language.

## Examples

Program that prints numbers from 100 to 1:

```
100;
ke;
while (soro > 0) {
    ke;
    print soro;
    soro - 1;
    ke;
}
```

## Compilation

```
cargo run -- com [-r] <filename.ceya>
```

`-r` to instantly run your program after compilation.
